# Bill of Materials

This page contains a brief description of and links for the commercially available parts needed to produce this platform   

# Wheel platform

* Structural: V-slot aluminium [profile](http://ooznest.co.uk/V-Slot/Linear-Rails/V-Slot-Linear-Rail-2020mm):
  - 290mm x 2
  - 300mm x 2
  - 260mm x 1
  - 120mm x 2
  - 180mm x 1
* Structural: M5 x 12, 8 and 6mm [screws](http://ooznest.co.uk/Low-Profile-M5-Bolts?search=M5) with accompanying [drop-in tee nuts](http://ooznest.co.uk/Drop-In-Tee-Nuts?search=drop)
* Structural: Universal (single) L-[bracket](http://ooznest.co.uk/V-Slot/Brackets/Universal-L-Brackets)
* Structural: 2-hole [joining plate](http://ooznest.co.uk/V-Slot/Brackets/2-Hole-Joining-Plate) x 2
* Structural: [Acrylic](http://www.acrylic-online.co.uk/) sheet 4mm (600 x 500mm minimum) x 1
* Structural: Assorted 3D printed [parts](https://github.com/croningp/NanoBot_hardware_V1/tree/master/STL%20files)
* Structural: Assorted M3 x 12/16/20/30mm [screws](https://www.accu.co.uk/en/64-low-head-cap-screws#elasticsearch_id_feature_655793=655793_697&elasticsearch_id_feature_655794=655794_1&id_elasticsearch_category=64&orderby=ranking&orderway=asc)
* Electronic: Arduino mega 2560 and [RAMPS](http://ooznest.co.uk/RAMPS-14-Controller-Board-Premium?search=ramps) 1.4 with 2 DRV8825. The options for the Arduino and stepper motor drivers are included in the drop down menu options x 1
* Electronic: 25 x 25 x 10 mm DC [fans](http://ooznest.co.uk/12V-DC-25mm-Fan?search=25mm) for the stirring array x 15
* Electronic: Corsair SF450 [power supply](https://www.amazon.co.uk/Corsair-SF450-Fully-Modular-Supply/dp/B01CR5XJR6) x 1
* Electronic: Nema17 stepper x 1 [motor](https://www.accu.co.uk/en/nema-17-stepper-motors/496466-NEMA17-40-1-7-1-8?uk_google_shopping=1&c=3&mkwid=s_dc&pcrid=209417175255&kword=&match=&plid=&gclid=CjwKCAjwyrvaBRACEiwAcyuzRABbU2tFQi1yQNd7_wL3WTq4IH_Au6toDr7JEU6PAY236qg2CjXwfBoCHX8QAvD_BwE) with accompanying Pololu aluminium mounting [hub](https://www.pololu.com/product/1998) (this part joins the drive wheel to the stepper motor shaft) x1
* Electronic: Any 2 and 4 way crimp housings. It matters that the crimps match the plastic housings you chose, beyond that many types can be used.
* Electronic: Stepper motor to RAMP connection [housings](https://uk.rs-online.com/web/p/products/2964956/?grossPrice=Y&cm_mmc=UK-PLA-DS3A-_-google-_-PLA_UK_EN_Connectors-_-Pcb_Connectors%7CPcb_Connector_Housings-_-PRODUCT+GROUP&matchtype=&gclid=CjwKCAjwyrvaBRACEiwAcyuzROPybPC5snwsWROQbjWMcLjBqjNpqaopkuRLn16-PZd7FsV7mNAtghoCZcMQAvD_BwE&gclsrc=aw.ds) x 1 per stepper motor (use with appropriate crimps for this housing)
* Electronic: End stop switch (modular syringe driver) crimp housings for RAMPS [connection](https://www.molex.com/molex/products/datasheet.jsp?part=active/0050579003_CRIMP_HOUSINGS.xml) (use with appropriate crimps for this housing)
* Pumps: Tricontinent [c-series pumps](https://www.tricontinent.com/c-series-syringe-pumps.html) x 8
* Pumps: Tubing and connectors from Kinesis/Cole-Palmer:
  - [1.6mm](https://kinesis.co.uk/tubing-tubing-ptfe-1-16-1-6mm-od-x-0-8mm-id-20m-008t16-080-20.html) OD tubing
  - [3.2mm](https://kinesis.co.uk/tubing-tubing-ptfe-1-8-3-2mm-od-x-1-5mm-id-10m-008t32-150-10.html) OD tubing
  - IDEX flangeless fittings Nat PP [1/16in](https://www.idex-hs.com/store/fluidics/fluidic-connections/fittings/flat-bottom-fittings/flangeless-fittings/flangeless-fitting-natural-polypropylene-1-4-28-flat-bottom-for-1-16-od.html)
  - IDEX flangeless fittings Nat PP [1/8](https://www.idex-hs.com/store/fluidics/fluidic-connections/fittings/flat-bottom-fittings/flangeless-fitting-natural-polypropylene-1-4-28-flat-bottom-for-1-8-od.html)
  - Restek Thames 1/4-28 female to male [Luer]() asy
  - Adhesive Dispensing (AD5125TLC) 25 gauge TLC lined tip [needle]()
* Fan stirring [magnets](https://www.first4magnets.com/rectangular-magnets-c35/15-x-4-x-4mm-thick-n42-neodymium-magnet-2-2kg-pull-p3499#ps_1-789) x 30
